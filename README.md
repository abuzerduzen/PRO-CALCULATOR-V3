# Pro Calculator v3

Pro Calculator v3, modern arayüze sahip, tema destekli bir hesap makinesi uygulamasıdır.  
Uygulama HTML, CSS ve JavaScript kullanılarak sıfırdan geliştirilmiştir.

## Proje Özellikleri
- Temel matematiksel işlemler (toplama, çıkarma, çarpma, bölme)
- Son karakteri silme (backspace) özelliği
- Yüzde hesaplama fonksiyonu
- Koyu ve açık tema desteği
- Hatalı operatör girişlerinin engellenmesi
- Glassmorphism (buzlu cam) tasarım yaklaşımı
- Mobil ve masaüstü cihazlarla uyumlu responsive yapı

## Kullanılan Teknolojiler
- HTML5
- CSS3  
  - CSS Grid  
  - CSS Variables  
  - Glassmorphism tasarım tekniği
- JavaScript (Vanilla JavaScript)

## Ekran Görüntüsü
Proje arayüzüne ait ekran görüntüsü ilerleyen aşamalarda eklenecektir.

## Teknik Not
Bu proje öğrenme amaçlı geliştirilmiştir.  
Hesaplama işlemlerinde JavaScript `eval()` fonksiyonu kullanılmıştır.
